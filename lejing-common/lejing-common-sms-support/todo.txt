lejing-common-sms-support:短信支持模块

核心注解:
@Sms(name="短信模板名称", platform=SmsEnum.XXX)

核心枚举：
SmsEnum{
 阿里云，
 华为云,
 七牛云,
 京东云,
 ;
}

短信模板方法顶层接口:
SmsTemplate {
 /**
  * 发送短信
  *
  * @param content 短信内容：验证码、字符内容等
  * @param phone 手机号：验证码、字符内容等
  */
  Object send(String... phone,String content);
}


短信模板方法实现类：
AliCloudSmsTemplateImpl implements SmsTemplate{}

QiniuCloudSmsTemplateImpl implements SmsTemplate{}

HuaweiCloudSmsTemplateImpl implements SmsTemplate{}

JingdongCloudSmsTemplateImpl implements SmsTemplate{}

使用方式：

@Resource
private SmsTemplate smsTemplate;

业务方法标注：
@PostMapping("/send/check/code")
@Sms(name="", platform=SmsEnum.XXX)
public void sendCheckCode(String... phone){
  String content = "10086";

 // 你的业务...

   Object response = smsTemplate.send(phone,content);
}
